# Wordle_Game
___

## About Project
Проект представляет из себя **Telegram bot** в котором воссоздана игра **Wordle** с понятным интерфейсом и всеми рабочими функциями.

В проекте есть возможность зайти как админ если вместо нажатия на кнопку войти ввести админский код ```Im the admin 1234```. Этот код ***хешируется***, так что в коде есть только хешированная строка, ***но не пароль***.

Все правила игры прописаны в ***описании бота***, котороевидно всем при запуске бота. Раз в сутки в 12 часов все старые слова обновляются, а новые слова создаются.

Всего за день можно решить *2* слова. Первое генерируется автоматически в 12:00 каждый день, второе может по желанию добавить любой из админов (***но тольно одно в день***).

## Installation
### Клонировать репозиторий
```python
https://github.com/Michail3078/Wordle_Game.git
```
### Установить зависиммости
```python
pip install -r requirements.txt
```
### запустите код
```python
python main.py
```
Рекомендуется версия ```python3.13```.

При запуске проекта запускайте файл ```main.py```. При запуске других файлов проект ***не будет*** полноценно работать.
